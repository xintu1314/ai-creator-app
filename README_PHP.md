# PHP版本使用说明

## 项目结构

```
app/
├── index.php              # 主入口文件
├── start.php              # PHP服务器启动脚本
├── includes/              # PHP组件文件
│   ├── sidebar.php
│   ├── header.php
│   ├── creation_area.php
│   ├── template_cards.php
│   └── dialogs.php
├── assets/                # 静态资源
│   ├── css/
│   │   └── style.css      # 自定义样式
│   └── js/
│       └── main.js        # JavaScript交互
└── src/                   # React版本（保留，可选）
```

## 启动方式

### 方式1：使用PHP内置服务器（推荐）

```bash
# 方法1：使用启动脚本
php start.php

# 方法2：直接启动
php -S localhost:8000
```

然后在浏览器访问：http://localhost:8000

### 方式2：使用Apache/Nginx

将项目目录配置为Web服务器根目录，访问对应域名即可。

- **后端**: PHP 7.4+
- **前端**: HTML + Tailwind CSS + JavaScript
- **图标**: Lucide Icons

## 界面效果

PHP版本与React版本界面效果**完全一致**，包括：
- 相同的颜色和布局
- 相同的动画效果
- 相同的交互体验
- 相同的响应式设计

## 注意事项

1. 确保PHP版本 >= 7.4
2. 如果使用PHP内置服务器，默认端口是8000
3. 所有样式和交互都已实现，可以直接使用
4. 生成功能需要连接后端API（可以后续添加）

## 开发建议

如果需要添加后端API：

1. 创建 `api/` 目录
2. 实现API接口（如 `api/generation/create.php`）
3. 在 `assets/js/main.js` 中取消注释API调用代码
4. 配置CORS和错误处理
